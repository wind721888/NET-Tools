# .NET Tools - 开发常用工具类汇总

## 📖 项目介绍

本项目是一个开源的 .NET 工具类集合，旨在整理、搜集 .NET 开发过程中高频使用的工具方法，帮助开发者减少重复编码工作。

## 🎯 核心目标

- 汇总高频工具类，避免开发者重复造轮子

- 保持代码简洁、规范，注释清晰，易于理解和使用

- 尽量适配 .NET 最新版本，兼容跨平台开发需求

- 开源共建，欢迎开发者提交常用工具类，完善项目生态

## 🚀 贡献指南

本项目欢迎所有 .NET 开发者参与贡献，一起完善工具类集合，让更多人受益！

### 贡献规范

- 工具类需聚焦 .NET 开发高频场景，避免冗余

- 代码简洁、高效，避免过度封装，保证易用性

- 不提交与项目无关的代码、文件（如：个人测试文件、无关依赖）

## 📌 注意事项

- 工具类均经过基础验证，但建议在生产环境使用前，根据自身业务场景再次测试

- 若发现工具类存在 Bug 或优化空间，可提交 Issue 或 Pull Request 反馈

- 禁止将本项目工具类用于违法、违规场景，使用本项目产生的一切风险由使用者自行承担
